Calculadora de Diárias de Hotel

Projeto desenvolvido para a disciplina de Software Básico na UMC.
O sistema calcula o valor total da hospedagem baseada em uma tabela de preços dinâmica (Pessoas x Tipo de Apartamento).

Funcionalidades:
Matriz de Preços: Utilização de Dicionários Python para mapear valores.
Validação de Dados: Tratamento de exceções (Try/Except) para inputs inválidos.
Interface: Uso de códigos ANSI para cores e limpeza de tela.
